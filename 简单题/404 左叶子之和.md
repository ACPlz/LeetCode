# 题目描述
## 计算给定二叉树的所有左叶子之和。
### 我的题解：
```class Solution {
public:
    int sumOfLeftLeaves(TreeNode* root) {
        queue<TreeNode*> que1;
        int nodeSum = 0;
        int count = 0;
        if (root == nullptr) return nodeSum;
        que1.push(root);
        while (!que1.empty()) {
            int size = que1.size() - count;
            for (int i = 0; i < size; i++) {
                TreeNode* node = que1.front();
                que1.pop();
                if ((que1.front() == nullptr) && (node->left == nullptr) &&(node->right == nullptr)) {
                    nodeSum += node->val;
                  //  printf("%d ", nodeSum);
                    que1.pop();
                    count--;
                }
                if ((que1.front() == nullptr) && ((node->left != nullptr) || (node->right != nullptr))) {
                   // printf("%d ", nodeSum);
                    que1.pop();
                    count--;
                }
                if (node->left) {
                    que1.push(node->left);
                    que1.push(nullptr);
                    count++;
                }
                if (node->right) {
                    que1.push(node->right);
                }
            }
        }
        return nodeSum;
    }
};
```
### **备注**：计算左叶子之和，注意左叶子表示没有左右孩子的点
### 这题其实用深搜一下就出来了，刚开始做的时候不懂为啥脑抽写了广搜，半途中意识到要用深搜的时候又懒得写，然后用广搜强行AC了。广搜麻烦的就在于要在左节点中加一个标记。
