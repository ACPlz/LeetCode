# 题目描述
## 斐波那契数 （通常用 F(n) 表示）形成的序列称为 斐波那契数列 。该数列由 0 和 1 开始，后面的每一项数字都是前面两项数字的和。
## 给定 n ，请计算 F(n) 。
### 我的题解：
```class Solution {
public:
    int fib(int n) {
        vector<int> dp(n + 1);
        if (n <= 1) return n;
        dp[0] = 0;
        dp[1] = 1;
        for (int i = 2; i <= n; i++) {
            dp[i] = dp[i - 1] + dp[i - 2];
        }
        return dp[n];
    }
};
```
### **备注**：动态规划第一题，状态转移方程和初始值都告诉我们了。
